使用python简化工作
Kale
 2019-12-08 15:49:55

算是比较有空吧今天，想起以前在部门工作的时候，工作很繁杂，接触了python之后，发现很多事情其实可以变得很简单。

之前自己创建一个志愿者库的制度，志愿者答题，如果正确率足够高，则可以加入志愿者库。以前的方法很死板，就是将导出的xlsx文件，一个人一个人地对，这样效率太低也太浪费时间了。

也是锻炼一下自己吧，以后应该都是用web比较多了，所以用一下qt练练手熟悉一下，本来想着写一个脚本实现自动录时长，但是听说现在都已经是直接导入文件就可以自动录时长了，很方便。

主要是读取xlsx文件，然后根据判断每一项是否为正确答案即可，逻辑很简单啦。界面设计得也很简单，毕竟功能就很简单嘛。

提一点是没有把正确答案写进代码里，还是创建了一个文件用以存储正确答案，也是方便他们以后更改吧。

就这样啦，写这个blog其实就是想纪念一下，毕竟这个制度也是我提出的，总不能让他们因为一些无谓的工作浪费了太长时间了。
